# KevPack

At the moment only includes a marker + trail pack for Magic Mirrors in Castora. 
This pack does not make use of relogging to reset mirrors to gain bonus keys. 		
It provides a trail to every mirror and chest on Castora, with instructions on when and where to get more keys when you run out.

The pack makes use of information windows provided by the BHud Pathing module for instructions.
